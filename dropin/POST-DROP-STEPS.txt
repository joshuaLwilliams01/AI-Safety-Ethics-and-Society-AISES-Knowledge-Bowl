
After you drag these files into your repo (matching the folder paths), run:

pnpm -w add socket.io-client
pnpm -w add -F api nanoid helmet
# (If you deploy the API separate from web) set CORS_ORIGIN to your Vercel URL.

Environment:
  apps/web/.env.local
    NEXT_PUBLIC_API_URL=http://localhost:4000

  apps/api/.env.local
    PORT=4000
    CORS_ORIGIN=http://localhost:3000

Run locally:
  pnpm -w dev
Open:
  Web: http://localhost:3000
  API health: http://localhost:4000/health

Optional:
  - Add audio files to apps/web/public/sounds/ (see README-SOUNDS.txt)
  - Replace sample board data with your CSV importer later.
